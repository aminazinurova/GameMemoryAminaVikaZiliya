Описание:
Игра "Memory", написана с помощью графической библиотеки tkinter. Суть игры заключается в запоминании и нахождении одинаковых пар карточек. В игре есть возможность играть с другом, а также с ИИ. Разработка велась на версии Python 3.13.9
